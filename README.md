# Whisper Audio CLI

Transcribe audios y busca coincidencias por palabras clave o frases usando Whisper + Fuzzy.

## Requisitos
- Python 3.9+
- GPU (opcional pero recomendado)
- ffmpeg instalado y en el PATH
- pip o conda
